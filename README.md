# Studio V

Full-stack PHP/MySQL website for Studio V, a photo studio and mobile servicing business.

## Run with XAMPP

1. Copy or keep this folder inside your XAMPP `htdocs` directory.
2. Start **Apache** and **MySQL** from the XAMPP Control Panel.
3. Open `http://localhost/phpmyadmin`.
4. Create a database named `studio_v`.
5. Import `database/studio_v.sql`.
6. Visit `http://localhost/Studio%20V/`.

## Admin

- URL: `http://localhost/Studio%20V/admin/login.php`
- Email: `admin@studiov.local`
- Password: `admin123`

Change the default admin password after first login in a real deployment.

## Configure Contact Details

Edit `includes/config.php` to update the shop phone number, email, address, WhatsApp number, and Google Maps embed URL.
